About SEQr

SEQr is a steganography web application that enables you to hide secret messages in image and audio files

It supports retrieval of messages from image and audio files that contain a secret message

Supported Formats

  • Image:-
  • .jpg
  • .png
  • Audio:-
  • .mp3
  • .wav

Outputs

  • .png images
  • .wav and .mp3 audio files
